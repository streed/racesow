# mappool.txt  --  map rotation for the Warsow race server
#
# One map name per line (no .bsp extension). Lines starting with '#' and blank
# lines are ignored. At startup entrypoint.sh keeps only the maps that are
# actually installed; anything not found is skipped with a log line.
#
# If this file is empty (all comments), the server auto-rotates through EVERY
# installed map instead -- which, on a fresh image, is whatever ships in the
# base Warsow install.
#
# To install community maps, drop .pk3 packs into server/maps/ on the host
# (scripts/fetch-maps.sh mirrors the whole livesow.net collection there). The
# directory is mounted read-only at /warsow/maps_extra and entrypoint.sh
# symlinks the packs into the mod dir at container start (see
# docker-compose.yml and server/maps/README.md).
# Then uncomment the maps you want below. These are the most-played maps in
# the livesow race database:
#
# aurora-speed1
# aurora-sunday

# coldrun
# bull-highway
